# Enterprise-Network-Design-Subnetting-DHCP-RIPv2
This project simulates the role of a junior network or cybersecurity engineer tasked with building a scalable, secure, and logically segmented enterprise network for TechLink Solutions, a mid-sized organization. This project demonstrates hands-on capability in subnetting, Layer 3 routing, DHCP configs, logical design, and enterprise documentation.
